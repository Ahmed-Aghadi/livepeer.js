# useAsset

Hook for retrieving an [Asset](https://github.com/livepeer/livepeer.js/blob/main/packages/core/src/types/lpms.ts)
based on a unique identifier.

```tsx
import { useAsset } from '@livepeer/react';
```

## Usage

The following examples assume an Asset was created via `useCreateAsset` or directly via a provider's API/dashboard.

```tsx
import { useAsset } from '@livepeer/react';

function App() {
  const { data: asset } = useAsset(asset?.id);
}
```

## Return Value

The return value is partially based on [React Query](https://tanstack.com/query/v4/docs/reference/useQuery), with some return types
aggregated for simplicity.

```tsx
{
  data?: Asset,
  error?: Error,
  status: 'idle' | 'loading' | 'success' | 'error',
  isError: boolean,
  isFetched: boolean,
  isFetching: boolean,
  isIdle: boolean,
  isLoading: boolean,
  isRefetching: boolean,
  isSuccess: boolean,
  refetch: (options: RefetchOptions) => Promise<UseQueryResult>,
}
```

## Configuration

### assetId

Video asset identifier. Can also be a string passed as the only parameter.

```tsx {5}
import { useAsset } from '@livepeer/react';

function App() {
  const { data: asset } = useAsset({
    assetId,
  });
}
```
