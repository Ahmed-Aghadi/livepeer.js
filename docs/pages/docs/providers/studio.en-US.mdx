# Studio

The `studioProvider` configures the client with [Livepeer Studio](https://livepeer.studio)

```tsx
import { studioProvider } from 'livepeer/providers/studio';
```

## Usage

```tsx
import { createReactClient } from '@livepeer/react';
import { studioProvider } from 'livepeer/providers/studio';

const client = createReactClient({
  provider: studioProvider({ apiKey: 'yourStudioApiKey' }),
});
```

## Configuration

### apiKey

Your [CORS protected](https://docs.livepeer.studio/guides/livestreaming/api-key#cors) Studio API key.
If no Studio API key is provided, the provider will error.

```ts {5}
import { createReactClient } from '@livepeer/react';
import { studioProvider } from 'livepeer/providers/studio';

const client = createReactClient({
  provider: studioProvider({ apiKey: 'yourStudioApiKey' }),
});
```
