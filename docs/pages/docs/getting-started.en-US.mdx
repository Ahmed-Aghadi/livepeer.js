# Getting Started

## Installation

Install `livepeer` and `@livepeer/react`, and the `wagmi` and `ethers` peer dependencies.

## Quick Start

### 1. Configure providers

First, configure your desired providers to be used by livepeer.

```tsx
import { configureLPMS } from 'livepeer';
import { studioProvider } from 'livepeer/providers/studio';

const { provider } = configureLPMS(
  [studioProvider()],
);
```

Note: In a production app, it is recommended to configure an API key for the `studioProvider` as you will face rate-limiting.

[Read more about configuring providers](/docs/providers/configuring-providers)

### 2. Create a livepeer client

Next, create a livepeer `Client` instance using [`createClient`](/docs/client), and pass the result from `configureChains` to it.

```tsx
import { createLivepeerClient } from 'livepeer';

const client = createLivepeerClient({
  provider
});
```

[Read more about client configuration](/docs/client)

### 3. Wrap app with `LivepeerConfig`

Finally, wrap your app with the [`LivepeerConfig`](/docs/LivepeerConfig) component, passing `client` to it.

```tsx
import { LivepeerConfig } from 'livepeer';

function App() {
  return (
    <LivepeerConfig client={client}>
      <SomeComponents />
    </LivepeerConfig>
  )
}
```

### 4. You're good to go!

Use hooks! Every component inside the `LivepeerConfig` is now set up to use the livepeer hooks.

```tsx
import { useAccount, useConnect, useEnsName } from 'livepeer'
import { InjectedConnector } from 'livepeer/connectors/injected'

function Profile() {
  const { address, isConnected } = useAccount()
  const { data: ensName } = useEnsName({ address })
  const { connect } = useConnect({
    connector: new InjectedConnector(),
  })

  if (isConnected) return <div>Connected to {ensName ?? address}</div>
  return <button onClick={() => connect()}>Connect Wallet</button>
}
```

Want to learn more? Check out the [examples](/examples/connect-wallet) to learn how to use livepeer in real-world scenarios or continue on reading the documentation.
