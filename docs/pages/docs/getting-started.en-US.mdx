## Quick Start

### 1. Configure chains

First, configure your desired chains to be used by livepeer, and the providers you want to use.

```tsx
import { configureChains, chain } from 'livepeer'
import { publicProvider } from 'livepeer/providers/public'

const { chains, provider, webSocketProvider } = configureChains(
  [chain.mainnet, chain.polygon],
  [publicProvider()],
)
```

Note: In a production app, it is not recommended to only pass `publicProvider` to `configureChains` as you will probably face rate-limiting on the public provider endpoints. It is recommended to also pass an [`alchemyProvider`](/docs/providers/alchemy) or [`infuraProvider`](/docs/providers/infura) as well.

[Read more about configuring chains](/docs/providers/configuring-chains)

### 2. Create a livepeer client

Next, create a livepeer `Client` instance using [`createClient`](/docs/client), and pass the result from `configureChains` to it.

```tsx {14-18}
import {
  LivepeerConfig,
  createClient,
  configureChains,
  defaultChains,
} from 'livepeer'
import { publicProvider } from 'livepeer/providers/public'

const { chains, provider, webSocketProvider } = configureChains(
  [chain.mainnet, chain.polygon],
  [publicProvider()],
)

const client = createClient({
  autoConnect: true,
  provider,
  webSocketProvider,
})
```

[Read more about client configuration](/docs/client)

### 3. Wrap app with `LivepeerConfig`

Finally, wrap your app with the [`LivepeerConfig`](/docs/LivepeerConfig) component, passing `client` to it.

```tsx {9-11}
const client = createClient({
  autoConnect: true,
  provider,
  webSocketProvider,
})

function App() {
  return (
    <LivepeerConfig client={client}>
      <YourRoutes />
    </LivepeerConfig>
  )
}
```

### 4. You're good to go!

Use hooks! Every component inside the `LivepeerConfig` is now set up to use the livepeer hooks.

```tsx {5,6,7-9}
import { useAccount, useConnect, useEnsName } from 'livepeer'
import { InjectedConnector } from 'livepeer/connectors/injected'

function Profile() {
  const { address, isConnected } = useAccount()
  const { data: ensName } = useEnsName({ address })
  const { connect } = useConnect({
    connector: new InjectedConnector(),
  })

  if (isConnected) return <div>Connected to {ensName ?? address}</div>
  return <button onClick={() => connect()}>Connect Wallet</button>
}
```

Want to learn more? Check out the [examples](/examples/connect-wallet) to learn how to use livepeer in real-world scenarios or continue on reading the documentation.
